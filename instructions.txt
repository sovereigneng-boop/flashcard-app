지금 구조를 크게 변경해야 해.

1. 메인 화면에 카드 나열하는 방식 제거. 세트에 들어가면 테스트 모드 선택 화면이 나와야 해.

2. 테스트 모드 3가지:

[데일리 퀴즈]
- 오늘 생성된 카드만 출제
- 카드 한 장씩 보여주고, 클릭하면 뒷면(답) 공개
- 답 확인 후 O/X 체크. 맞추면 해당 카드는 이번 세션에서 제외
- 복습 회차와 무관. 그날 세션 내에서만 유효
- 모든 카드를 맞추면 데일리 퀴즈 완료

[데일리 복습]
- 복습주기에 해당하는 카드만 출제
- 카드 한 장씩 보여주고, 클릭하면 뒷면(답) 공개
- 답 확인 후 O/X 체크
- 맞추면 다음 회차로 진행
- 틀리면 회차 1 감소 + 다음날 다시 출제
- 반드시 맞춰야만 다음 회차로 넘어감. 안 풀면 계속 그 회차에 남아있음
- 카드 하단에 지금까지의 O/X 기록 표시
- 복습 주기 (맞춘 후 다음 출제일):
  1회차: 1일 후
  2회차: 3일 후
  3회차: 1주 후
  4회차: 2주 후
  5회차: 1달 후
  6회차: 졸업 (더 이상 출제 안됨)

[토탈 테스트]
- 해당 세트의 모든 카드 테스트 (복습주기 무관)
- O/X 체크, 카드 하단에 O/X 기록 표시

3. 카드 데이터 필드:
- front: 앞면 텍스트
- back: 뒷면 텍스트
- createdAt: 생성 날짜
- reviewStage: 현재 복습 회차 (1~6)
- nextReviewDate: 다음 복습 예정일
- history: O/X 기록 배열 (날짜 포함)

4. 편집 버튼은 유지 (표 형태로 카드 추가/수정/삭제)
5. 카드를 수정(편집)해도 createdAt은 변경하지 않음. 최초 생성 날짜를 유지해야 함.
